# 【全栈之旅】Node.js + Vue.js 开发web应用示例

安装 cnpm
```shell
  npm install -g cnpm --registry=https://registry.npm.taobao.org
```
如不安装可修改`client` AND `server`目录下package.json的scripts命令，将`cnpm install`修改为`npm install`即可

### 本地运行
 - 安装mysql
 - 安装node.js
#### 安装包依赖
```
  npm install
```
#### 启动
```
  npm run start
```
